### Задача ###

#### "Многопоточность" - Гонки ####

Количество автомобилей вводить с консоли (потоки), длину трассы тоже с консоли. Запускать одновременно. Каждую 
секунду логировать сколько до финиша. Автомобиль хранит в себе скорость.
